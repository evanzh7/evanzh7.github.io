python学习（一）——python文件处理

2014-07-30

open函数（name,[,mode[,buffering]])

  • name: 文件名
  • mode: 文件模式
  • buffering: 文件缓冲
  • 返回值：文件对象

##三种标准的流

  1. 数据输入的标准源————sys.stdin内
  2. 要打印的文本保存在————sys.stdout内
  3. 错误信息写入————sys.stderr

###1.读写文件步骤

f.write(‘hello world’);
//没有writeline()方法
f.writelines();

f.read()//不带参数读取所有
f.read(‘4’);//读取多少个字节
f.readline()
f.readlines();//读取一个文件所有行并将其作为列表返回

with语句可以打开文件，并且将其赋值到变量上（本例中是somefile）。文件在语句结束后会被自动关闭。即使是由于异常引起的结束也是如此。

###5.迭代文件
2.2版本之后，文件是可迭代的，下面是文件迭代方法

简化版的例子 

for line in open(filename)
process(line)
//sys.stdin是可迭代的  
import sys  
for line in sys.stdin:  
process(line)
