LATEX（通常读作/ˈleɪ.tɛk/（音译“莱泰克”）或者/ˈlɑ.tɛk/（音译“拉泰克”））文字形式写作LaTeX，是一种基于TEX的排版系统，由美国电脑学家莱斯利·兰伯特在20世纪80年代初期开发,

###MathJax

  1. 内联的公式用$...$来表示，单独一行展示公式用$$...$$他们的展示是不同的。
  2. 对于希腊字母，使用\alpha,\beta,…\omega:$\alpha$, $\beta$, …$\omega$. 对于大写字母，使用 \Gamma, \Delta, …,\Omega:$\Gamma$, $\Delta$,…$\Omega$来表示
  3. 对于指数位置和底数位置，使用^和_。例如，x_i^2: $x_i^2$。
  4. Groups. 指数，底数，只应用到紧挨着的下一组之中，一组既可以是一个符号，或者被中括号围起来的一个公式{...}。如何你这样写10^10,你会得到这样的结果：$10^10$。但是如果你这样写10^{10},编辑器就给你想要的结果$10^{10}$， 用括号能够限制符号应用的范围。例如：x^5^6是个错误，{x^y}^z就可以表示为${x^y}^z$, x^{y^z}就可以表示为$x^{y^z}$。观察x_i^2$xi^2$和`x{i^2}`$x_{i^2}$。

  5. 括号 普通的括号()[]就是括号的样子(2+3)[4+4]。使用转义字符\{和\}来表示花括号。{ } 。

    括号不会随着其中公式放大或缩小。如果你这样写(\frac12)括号就会太小：$(\frac12)$。用\left(…\right)将会使括号随着公式自动调整大小$\left(\frac12\right)$ 。
    \left和\right可以应用到下面这些种类的括号中

    • 小种类：
      1. (和) $\left(x\right)$,
      2. [和]$\left[x\right]$,
      3. \{和 \} ${x}$
      4. 竖括号| $| x |$
      5. 尖括号 \langle \rangle $\langle x \rangle$
      6. 天花板 \lceil \rceil $\lceil x \rceil$
      7. 地板 \lfloor \rfloor $\lfloor x \rfloor$
      8. 隐形的括号用.表示：\left.\frac12\right\rbrace $\left.\frac12\right\rbrace$

  6. 求和 积分 \sum和int下标是最低限度，上标是最高限度，例如\sum_1^n $\sum1^n$， 如果限制不仅仅是一个符号。不要忘记使用括号{...}，例如 $\sum{i=0}^\infty i^2$。 类似的，

    1. 求积用\prod $\prod$
    2. 求积分 \int $\int$
    3. 求并 \bigcup $\bigcup$
    4. 求交 、bigcap $\bigcap$
    5. 这是啥？ \iint $\iint$
  7. 分数
    三种形式
    1. 简单形式：\frac ab $\frac ab$
    2. 复杂形式(用中括号)：\frac{a+1}{b+1} $\frac{a+1}{b+1}$
    3. 最复杂形式(用\over):{a+1\over b+1} ${a+1\over b+1}$
  8. 字体
    这玩意出版行业的人用得到吧，不多说了
  9. 根号
    1. 默认二次根号： \sqrt{x^3} $\sqrt{x^3}$
    2. 三次根号：\sqrt[3]{\frac xy} $\sqrt[3]{\frac xy}$
    3. 更复杂形式用花括号{...}^{1/2}
  10. 特殊函数
    一些特殊符号例如lim sin max ln 通常是罗马字体， 而不是意大利字体，使用\lim \sin \lim_{x\to 0} $$\lim_{x\to 0}$$
  11. 特殊符号和标识
  12. 空格
  13. 重音符号和变音符号
  14. 特殊符号特殊字符用于MathJax解释可以使用\字符转义。
